# Recruitment AI Assistant

A Windows desktop application that uses AI to assess English proficiency from voice recordings and automatically match candidates with suitable job opportunities.

## üöÄ Features

- **Candidate Management**: Complete candidate profile system with quadruple name support
- **AI English Assessment**: Custom AI model to evaluate English levels (A1-C2 CEFR) from 45+ second voice notes
- **Automated Job Matching**: Intelligent job-candidate matching based on English proficiency and profile
- **WhatsApp Integration**: Automated candidate communication via WhatsApp
- **Google Sheets Sync**: Real-time synchronization with Google Sheets for HR management
- **Secure Access**: Role-based permissions (CEO, Manager, Recruiter levels)
- **Voice Note Processing**: Support for various audio formats (MP3, WAV, M4A, AAC)
